# Food Recipe App üçΩÔ∏è

A full-stack food recipe web application where users can discover, save, share, and rate recipes with a vibrant community of food lovers. Built using React.js for the frontend, Spring Boot for the backend API, and MySQL (SQL Workbench) for data storage.


## Features

- User authentication and account management (Sign-up, Login, Password recovery via registered email)
- Recipe discovery, detailed view with step-by-step instructions
- Explore regional and international cuisines
- Search recipes by name or location
- Rate and comment on recipes after trying them out
- Responsive, modern UI with rich food imagery

## Tech Stack

- **Frontend**: React.js
- **Backend**: Spring Boot
- **Database**: MySQL (SQL Workbench)
- **Email Integration**: Password reset functionality via registered email

## How It Works

1. **Sign up or Login:** New users can register. Forgotten passwords are easily recovered (sent to registered email).
2. **Browse Recipes:** Authenticated users can browse a variety of food items with images.
3. **Recipe Details:** Click any food item to see prep steps and watch video guides.
4. **Rate & Review:** Users can leave ratings/comments after cooking.
